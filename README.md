## Інструкція для завантаження всіх dependencies

1. Оновіть всі файли.
```bash
git pull
```
2. Налаштуйте `venv`.
```bash
python -m venv .venv
```
3. Активуйте
   * Windows (PowerShell): ``.\.venv\Scripts\activate``
   * Mac/Linux: ``source .venv/bin/activate``
4. Встановіть необхідні бібліотеки
```bash
pip install -r requirements.txt
```
